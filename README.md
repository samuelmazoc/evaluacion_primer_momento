## Descripción
Este proyecto carga un archivo CSV con compras, calcula estadísticas y genera un reporte JSON.

```bash
python main.py
```

## Estructura
- `data/compras.csv` → Datos de entrada.
- `main.py` → Programa principal.
- `utils.py` → Funciones auxiliares.
